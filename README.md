# Inventory Optimization & Stockout Analysis

End-to-end business-focused inventory optimization project designed to analyze stockouts, inventory turnover, and replenishment strategies.

## 🎯 Business Objective

Improve working capital efficiency and reduce lost sales by:

- Identifying stockout patterns
- Measuring inventory performance
- Designing safety stock logic
- Defining reorder point strategy
- Quantifying financial impact

---

## 📊 Core KPIs

- Stockout Rate
- Inventory Turnover
- Days of Inventory on Hand (DOH)
- Sell-Through Rate
- GMROI (Gross Margin Return on Inventory)
- Safety Stock Level
- Reorder Point

---

## 🛠 Tech Stack

- SQL (data cleaning, aggregations, KPI calculation)
- Python (inventory modeling, safety stock simulation)
- Power BI (executive dashboard)
- Star schema data modeling

---

## 📂 Repository Structure

- data/ → name and link of the dataset used
- sql/ → queries to clean and calculate KPIs
- python/ → code to analyze the inventory
- powerbi/ → inventory-dashboard.pbix

---

## 📈 Project Workflow

1. Business case definition
2. KPI framework design
3. Data exploration
4. SQL KPI calculations
5. Safety stock & reorder point modeling
6. Dashboard development
7. Executive recommendations

---

## 🚀 Business Impact (To Be Measured)

- Reduced stockout rate
- Improved inventory turnover
- Optimized safety stock levels
- Enhanced capital allocation
